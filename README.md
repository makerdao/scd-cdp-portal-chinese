## CDP Portal

### Prerequisites

Have installed:

- Git
- Node 8 or higher

### Installation

1) `git clone git@github.com:makerdao/scd-cdp-portal-chinese.git`

2) `cd scd-cdp-portal-chinese && yarn`

### Run project

- `yarn start`
- go to http://localhost:3000 (default port)

For HW support:
- `HTTPS=true yarn start`
- go to https://localhost:3000 (default port)
